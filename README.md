## 프로젝트
### 소개
  첫 팀 프로젝트입니다. 라이브러리와 오픈 API를 활용해 팀작업을 경험해보라는 요구사항이 있었습니다.
### 주요 구현
  - formik, yup 라이브러리를 활용한 로그인, 회원가입을 구현.
  - 카카오 로컬, Map API를 활용해 검색창에 키워드 검색 시 검색한 위치로 지도 이동 구현.
  - 키워드 관련 주변 목록 출력, 목록 클릭 시 해당 위치로 지도 이동 구현.
